# ODE Solver with Runge-Kutta in Python

This repository contains Python implementations of numerical methods for solving ordinary differential equations, with emphasis on Runge-Kutta methods.

## Main goals
- Implement classical Runge-Kutta methods in Python
- Compare numerical solutions with analytical solutions when available
- Analyze numerical error
- Visualize solutions and convergence behavior

## Planned contents
- RK2 and RK4 implementations
- Example initial value problems
- Error analysis
- Plots of numerical and exact solutions
- Clean and well-documented Python scripts

## Tools
- Python
- NumPy
- Matplotlib
